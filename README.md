## Dependencies
1. imgui-Docing
2. glm
3. spdlog
4. glfw
5. glew
6. SOIL2
7. tinyxml

## Features
- ECS FrameWork
  - Components
    - Transfoem
    - DirectionLight
    - PointLight
    - Renderer
    - Camera
    - Behaviour
  - System
    - RenderSystem
- Base components of game engine
  - Reflection for c++
  - A debug tool made with spdlog
  - A editor windows made with imgui  
  - (Only opengl now)Render framework supporting the multiple graphics API
